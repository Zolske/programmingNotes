the dom

  • When a web page is loaded, the browser creates a Document Object Model of the page.
  • The DOM is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document (the original document is not changed only its model).

  • The W3C DOM standard is separated into 3 different parts:
    1. Core DOM - standard model for all document types
    2. XML DOM - standard model for XML documents
    3. HTML DOM - standard model for HTML documents

  • html elements are represented as nodes, those nodes them self can have "attribute" nodes and "text" nodes
  • the content of the DOM can be accessed by JS over the "document object". Which provides methods to change, read and create nodes in the DOM
  • the DOM is represented as a tree structure with the "root" node at the top from which the "child" nodes ascent.

accessing the node element

  • First, the "document-object" needs to be accessed over the method document.
  • After, the "element-node" can be accessed (can be written on one line).
    • Evaluates to the "element-node" and its child-nodes if it has any:
      • "text-node" if it has any
      • "attribute-node" if it has any
  • Evaluates to null if no element-node is found.

access by id

document.getElementById(' ')
  • takes as an argument the id of the html element as a string

access by tag name

document.getElementsByTagName(' ')
  • Takes as an argument the html element tag name (e.g. <p>) as a string without < > (e.g. "p").
  • All elements with the referenced tag are combined into one array, even if there is only one element, remember to use the [index] to access the elements individually.
  • let pElement = document.getElementsByTagName("p");
    //get all the <p> elements and assign it to the variable "pElement"
    • console.log(pElement);
    • console.log(pElement[0]); //selects the first element of the "pElement" array
    • .length evaluates to the length of the array

Accessing text-node and attribute-node

Can be only accessed over the "element-node". Remember, that the "...ByTagName" is an array, a specific element needs to be selected through the index.

access text-element

Element.Node.firstChild.nodeValue
  • Evaluates to the "text-node" of the "element-node", which is a string.
  • let textNode = Element_Node.firstChild.nodeValue;
    //saves the value of the "text-node" to the variable textNode
  • Element_Node.firstChild.nodeValue = 'new text';
    //assigns 'new text' string to the "text-node".

access attribute-node

Element.Node.getAttribute('style')
  • Evaluates to the "attribute-node" of the "element-node", which is a string.
  • style evaluates to a string which contains the value of the style attribute (color: red;)
