class

Different objects with the same class are related, they are members. They all have the same properties but can have different property-values. The methods which are defined in the class, are accessible by all its members.

accessing the property of an object

dot notation
object.propertyKey evaluates to the property-value of the object
object object name of the property
propertyKey name of the property-key, needs to follow the naming convention for variable names
object.property() accesses the method of the object (arguments can be passed inside ( ) )
bracket notation
object['propertyKey'] evaluates to the property-value of the object
object object name of the property
propertyKey is just a string (or Symbol) of the property-key, dose not need to follow the naming convention for variable names. The string can be created trough expressions or variables.
object['propertyKey']() accesses the method of the object (arguments can be passed inside ( ) )

creating a reference to an object and comparing

When an object is crated, it is assigned to a (original) variable which references the object. Assigning the (original) variable to another variable dose not create a new object, it just references the same object. Because it is the same object which is referenced, it dose not matter over which variable the object is changed.

Can be used as condition in an if statements, ( variable_1 OPERATOR variable_2 )

==
true both variable reference the same object
false do not reference the same object
!=
true both variable do not reference the same object
false reference the same object

A variable which references an object, can be passed as an argument to an function. All its properties and method can be changed or accsessed within the function.

JSON

JavaScript Object Notation is a compact form of writing objects (with there key-value pairs) and arrays in string form together. It makes it easier to exchange data.

json keyword object object keyword json
VARIABLE = JSON.stringify (VARIABLE) ; VARIABLE = JSON.parse (VARIABLE) ;
variable in which the string is saved variable references the object from which a copy will be transformed into a string variable will reference the object, which will be crated from the json string variable which holds the json string 
    let testObject = {
        key1 : 'value1',
        key2 : 'value2'
      };
      
      console.log(testObject);                      //{ key1: 'value1', key2: 'value2' }
      
      let testJson = JSON.stringify(testObject);
      
      console.log(testJson);                        //'{"key1":"value1","key2":"value2"}'
      
      console.log(JSON.parse(testJson))             //{ key1: 'value1', key2: 'value2' }
