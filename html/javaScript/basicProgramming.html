The basic in programming

variable

A variable reserves and references space in memory. Data can be stored and recalled from that space. The name of the variable is than evaluated to the data which is stored at its location.
The type of the data dose not need to be defined before and can change when the variable is updated.

naming convention

  • can not start with a number
  • can not contain special characters (ß, ä) , empty space, hyphen
  • can not be a key- or reserved- word
  • can contain small / capital letters, numbers, underscore
  • the name should convey the purpose and be notated in camelCase

performing operations on strings
JavaScript provides properties and methods to perform operations on string values. Technically, only objects can have properties and methods. But in JavaScript primitive data types can act like objects when you refer to them with the property access notation (dot notation).
concatenating two or more strings
+
+=		 You can concatenate or combine two or more strings using the + and += assignment operators. 
var hello = "Hello";
var world = "World";
var greet = hello + " " + world;
document.write(greet); // Prints: Hello World
 
var wish  = "Happy";
    wish += " New Year";
document.write(wish); // Prints: Happy New Year
accessing individual characters from a string
VARIABLE[INDEX]
VARIABLE.charAt(INDEX)		 The index determines the position of the to return letter from the string. 
var str = "Hello World!";
document.write(str.charAt());  // Prints: H
document.write(str.charAt(6)); // Prints: W
document.write(str.charAt(30)); // Prints nothing
document.write(str.charAt(str.length - 1)); // Prints: !

var str = "Hello World!";
document.write(str[0]); // Prints: H
document.write(str[6]); // Prints: W
document.write(str[str.length - 1]); // Prints: !
document.write(str[30]); // Prints: undefined
getting the length of a string
VARIABLE.length Returns the length of the string, which is the number of characters contained in the string (including special characters such as \t or \n). 
var str1 = "This is a paragraph of text.";
document.write(str1.length); // Prints 28
 
var str2 = "This is a \n paragraph of text.";
document.write(str2.length); // Prints 30, because \n is only one character
finding a string inside another string
VARIABLE.indexOf('SUBSTRING')
VARIABLE.indexOf('SUBSTRING', INTEGER)		 Returns the index of the first occurrence of a specified (sub) string within a string. Returns -1 if not found.
INTEGER the number specifies from where to start to look for the SUBSTRING.
var str = "If the facts don't fit the theory, change the facts.";
var pos = str.indexOf("facts");
alert(pos); // 0utputs: 7
VARIABLE.lastIndexOf('SUBSTRING')
VARIABLE.lastIndexOf('SUBSTRING', INTEGER) 		Returns the index of the last occurrence of the (sub) specified string within a string. Returns -1 if not found.
INTEGER the number specifies from where to start to look for the SUBSTRING.
var str = "If the facts don't fit the theory, change the facts.";
var pos = str.lastIndexOf("facts");
alert(pos); // 0utputs: 46
extracting a substring from a string
VARIABLE.slice(startINDEX, endINDEX) Takes 2 parameters: start index (index at which to begin extraction), and an optional end index (index before which to end extraction). If no end index than the rest of the string is returned. 
var str = "The quick brown fox jumps over the lazy dog.";
var subStr = str.slice(4, 15);
document.write(subStr); // Prints: quick brown
A negative parameter is counted backwards from the end of the string. 
var str = "The quick brown fox jumps over the lazy dog.";
document.write(str.slice(-28, -19)); // Prints: fox jumps
document.write(str.slice(31)); // Prints: the lazy dog.
replacing the contents of a string
VARIABLE.replace('oldSTRING', 'newSTRING') This replace() method returns a new string, it doesn't affect the original string that will remain unchanged. Replaces part of a string with another string. This method takes two parameters, oldSTRING is the string which will be replaced with newSTRING. 
var str = "Color red looks brighter than color blue.";
var result = str.replace("color", "paint");
alert(result); // 0utputs: Color red looks brighter than paint blue.
converting a string to uppercase or lowercase
VARIABLE.toUpperCase() Convert a string to uppercase, no parameter. 
var str = "Hello World!";
var result = str.toUpperCase();
document.write(result); // Prints: HELLO WORLD!
VARIABLE.toLowerCase() Convert a string to lowercase, no parameter.

the number data type

  • Can be represented in decimal (2.1), hexadecimal (0xff) or octal notation.
  • JavaScript does not treat integer (2) and floating-point (3.14) numbers differently.
  • All numbers in JavaScript are represented as floating-point numbers.
  • exponential notation: 4.25e+6 = 4250000 and 4.25e-6 = 0.00000425

addition on numbers and strings

  • If a sting is part of an addition, than the numbers are converted to strings.
    var x = 10;
var y = 20;
var z = "30";

// Adding a number with a number, the result will be sum of numbers
console.log(x + y); // 30

// Adding a string with a string, the result will be string concatenation
console.log(z + z); // '3030'

// Adding a number with a string, the result will be string concatenation
console.log(x + z); // '1030'

// Adding a string with a number, the result will be string concatenation
console.log(z + x); // '3010'

// Adding strings and numbers, the result will be string concatenation
console.log("The result is: " + x + y); // 'The result is: 1020'
  • Operations a solved from left to right, two or more numbers are added before if a string is coming after.
    var x = 10;
var y = 20;
var z = "30";

// Adding numbers and strings, calculation performed from left to right
console.log(x + y + z); // 'The result is: 3030'

operating on numbers and strings (except addition)

  • String is converted to a number. 
    var x = 10;
var y = 20;
var z = "30";

// Subtracting a number from a number
console.log(y - x); // 10

// Subtracting a number from a numeric string
console.log(z - x); // 20

// Multiplying a number with a numeric string
console.log(x * z); // 300

// Dividing a number with a numeric string
console.log(z / x); // 3
  • If string can not be converted to a number, NaN (Not a Number) is returned.
    var x = 10;
var y = "foo";
var z = NaN;

// Subtracting a number from a non-numeric string
console.log(y - x); // NaN

// Multiplying a number with a non-numeric string
console.log(x * y); // NaN

// Dividing a number with a non-numeric string
console.log(x / y); // NaN

// Adding NaN to a number 
console.log(x + z); // NaN
                        
// Adding NaN to a string 
console.log(y + z); // fooNaN

representing infinity

  • Infinity represents a number too big for JavaScript to handle.
  • JavaScript has special keyword Infinity and -Infinity to represent positive and negative infinity respectively. 
    var x = 5 / 0;
console.log(x); // Infinity

var y = -5 / 0;
console.log(y); // -Infinity

Avoiding Precision Problems (representation error)

  • JavaScript and many other languages uses binary (base 2) form to represent decimal (base 10) numbers internally. Unfortunately, most decimal fractions can't be represented exactly in binary form, so small differences occur. 
    var x = 0.1 + 0.2;
console.log(x) // 0.30000000000000004
  • To avoid this problem, the fractions can be multiplied by a factor so they become "whole" numbers, operated on and than divided by the same factor.
    var x = (0.1 * 10 + 0.2 * 10) / 10;
console.log(x) // 0.3
  • Floating-point numbers are rounded up if longer than 16 digits after the floating pint ..
  • Integers (numbers without fractional parts or exponential notation) are rounded up if longer than 15 digits.
    var x = 999999999999999;
console.log(x); // 999999999999999

var y = 9999999999999999;
console.log(y); // 10000000000000000
math operations (objects)
syntax description example
Math.abs() Calculate the absolute (positive) value of a number. Therefore, -1 is returned as 1. 
document.write(Math.abs(-1));  // Prints: 1
document.write(Math.abs(1));  // Prints: 1
document.write(Math.abs(-5));  // Prints: 5
document.write(Math.abs(-10.5));  // Prints: 10.5
Math.random() Generate a floating-point random number between 0 and 1 (but not 1). 
Math.random();  // Expected output: a number between 0 and 1
// Could be 0.46722224
Math.floor(Math.random() * MAX) Generates a random number between 0 and MAX-1. 
Math.floor(Math.random() * 3) // Expected output: 0, 1 or 2
Math.floor(Math.random() * 1) // Expected output: 0
Math.sqrt() Calculates the square root of a number. If the number is negative, NaN is returned. 
Math.sqrt(4);  // Prints: 2
Math.sqrt(16);  // Prints: 4
Math.sqrt(0.25);  // Prints: 0.5
Math.sqrt(-9);  // Prints: NaN
Math.ceil() Rounds a number up to the next highest integer. So, 3.5 becomes 4, -5.7 becomes -5 (because -5 is greater than -6). 
Math.ceil(3.5);  // Prints: 4
Math.ceil(-5.7);  // Prints: -5
Math.ceil(9.99);  // Prints: 10
Math.ceil(-9.99);  // Prints: -9
Math.ceil(0);  // Prints: 0
Math.floor() Rounds a number down to the next lowest integer. So, 3.5 becomes 3, -5.7 becomes -6 (because -6 is lesser than -5). 
Math.floor(3.5);  // Prints: 3
Math.floor(-5.7);  // Prints: -6
Math.floor(9.99);  // Prints: 9
Math.floor(-9.99);  // Prints: -10
Math.floor(0);  // Prints: 0
Math.round() Rounds a number to the nearest integer in such a way that if the decimal part is .5 or greater, number is rounded up, otherwise rounded down. So, 3.5 becomes 4, -5.7 becomes -6, 4.49 becomes 4 
Math.round(3.5);  // Prints: 4
Math.round(-5.7);  // Prints: -6
Math.round(7.25);  // Prints: 7
Math.round(4.49);  // Prints: 4
Math.round(0);  // Prints: 0
Math.max()
Math.min()		 Finds which number is the largest or smallest in a group of numbers, respectively. It works also on arrays. 
document.write(Math.max(1, 3, 2));  // Prints: 3
document.write(Math.max(-1, -3, -2));  // Prints: -1
 
document.write(Math.min(1, 3, 2));  // Prints: 1
document.write(Math.min(-1, -3, -2));  // Prints: -3

var numbers = [1, 3, 2]; // works also on array
 
document.write(Math.max.apply(null, numbers));  // Prints: 3
document.write(Math.min.apply(null, numbers));  // Prints: 1

var numbers = [1, 3, 2]; // or with the spread operator ...
 
document.write(Math.max(...numbers));  // Prints: 3
document.write(Math.min(...numbers));  // Prints: 1
