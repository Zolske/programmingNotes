variable

A variable reference a specific place in memory. Data can be stored and recalled from that place. The name of the variable is than evaluated to the data which is stored at its location.

variable types

variable description
const
  • needs to be declared with an value
  • its value can not be changed after it has been declared
  • elements of an array or object can be still changed
let
  • dose not need to be declared with an value ("undefined")
  • const the data can not be changed
