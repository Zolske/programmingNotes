The basic in programming

variable

A variable reserves and references space in memory. Data can be stored and recalled from that space. The name of the variable is than evaluated to the data which is stored at its location.
The type of the data dose not need to be defined before and can change when the variable is updated.

naming convention

  • can not start with a number
  • can not contain special characters (ß, ä) , empty space, hyphen
  • can not be a key- or reserved- word
  • can contain small / capital letters, numbers, underscore
  • the name should convey the purpose and be notated in camelCase

performing operations on strings
JavaScript provides properties and methods to perform operations on string values. Technically, only objects can have properties and methods. But in JavaScript primitive data types can act like objects when you refer to them with the property access notation (dot notation).
concatenating two or more strings
+
+=		 You can concatenate or combine two or more strings using the + and += assignment operators. 
var hello = "Hello";
var world = "World";
var greet = hello + " " + world;
document.write(greet); // Prints: Hello World
 
var wish  = "Happy";
    wish += " New Year";
document.write(wish); // Prints: Happy New Year
accessing individual characters from a string
VARIABLE[INDEX]
VARIABLE.charAt(INDEX)		 The index determines the position of the to return letter from the string. 
var str = "Hello World!";
document.write(str.charAt());  // Prints: H
document.write(str.charAt(6)); // Prints: W
document.write(str.charAt(30)); // Prints nothing
document.write(str.charAt(str.length - 1)); // Prints: !

var str = "Hello World!";
document.write(str[0]); // Prints: H
document.write(str[6]); // Prints: W
document.write(str[str.length - 1]); // Prints: !
document.write(str[30]); // Prints: undefined
getting the length of a string
VARIABLE.length Returns the length of the string, which is the number of characters contained in the string (including special characters such as \t or \n). 
var str1 = "This is a paragraph of text.";
document.write(str1.length); // Prints 28
 
var str2 = "This is a \n paragraph of text.";
document.write(str2.length); // Prints 30, because \n is only one character
finding a string inside another string
VARIABLE.indexOf('SUBSTRING')
VARIABLE.indexOf('SUBSTRING', INTEGER)		 Returns the index of the first occurrence of a specified (sub) string within a string. Returns -1 if not found.
INTEGER the number specifies from where to start to look for the SUBSTRING.
var str = "If the facts don't fit the theory, change the facts.";
var pos = str.indexOf("facts");
alert(pos); // 0utputs: 7
VARIABLE.lastIndexOf('SUBSTRING')
VARIABLE.lastIndexOf('SUBSTRING', INTEGER) 		Returns the index of the last occurrence of the (sub) specified string within a string. Returns -1 if not found.
INTEGER the number specifies from where to start to look for the SUBSTRING.
var str = "If the facts don't fit the theory, change the facts.";
var pos = str.lastIndexOf("facts");
alert(pos); // 0utputs: 46
extracting a substring from a string
VARIABLE.slice(startINDEX, endINDEX) Takes 2 parameters: start index (index at which to begin extraction), and an optional end index (index before which to end extraction). If no end index than the rest of the string is returned. 
var str = "The quick brown fox jumps over the lazy dog.";
var subStr = str.slice(4, 15);
document.write(subStr); // Prints: quick brown
A negative parameter is counted backwards from the end of the string. 
var str = "The quick brown fox jumps over the lazy dog.";
document.write(str.slice(-28, -19)); // Prints: fox jumps
document.write(str.slice(31)); // Prints: the lazy dog.
replacing the contents of a string
VARIABLE.replace('oldSTRING', 'newSTRING') This replace() method returns a new string, it doesn't affect the original string that will remain unchanged. Replaces part of a string with another string. This method takes two parameters, oldSTRING is the string which will be replaced with newSTRING. 
var str = "Color red looks brighter than color blue.";
var result = str.replace("color", "paint");
alert(result); // 0utputs: Color red looks brighter than paint blue.
converting a string to uppercase or lowercase
VARIABLE.toUpperCase() Convert a string to uppercase, no parameter. 
var str = "Hello World!";
var result = str.toUpperCase();
document.write(result); // Prints: HELLO WORLD!
VARIABLE.toLowerCase() Convert a string to lowercase, no parameter. 
var str = "Hello World!";
var result = str.toLowerCase();
document.write(result); // Prints: hello world!

the number data type

  • Can be represented in decimal (2.1), hexadecimal (0xff) or octal notation.
  • JavaScript does not treat integer (2) and floating-point (3.14) numbers differently.
  • All numbers in JavaScript are represented as floating-point numbers.
  • exponential notation: 4.25e+6 = 4250000 and 4.25e-6 = 0.00000425
