conditional statements

  • Condition evaluates to true or truthy, the code inside the block is executed.
  • Condition evaluates to false or falsy, the code inside the block is NOT executed.
  • A default code block can be run if the condition is false / falsy.

if statement

  • "if - else if - else", statements are parsed from top to bottom.
  • Only the first "true-condition-statement-block" is executed, any following conditions are not checked!
  • No brackets {} are needed if there is only one statement after the condition.
syntax explanation example
if statement
if (CONDITION) STATEMENT; or
if (CONDITION) {STATEMENT; STATEMENT2;}
  • If the condition evaluate to true / truthy, than the statements are executed.
  • The "if statement" can stand by its own.
 
const a = 12, b = 7;

if(a > b) // condition is true, statement is executed 
   document.write("<p>a is bigger than b</p>");
else if statement (optional)
else if (CONDITION) STATEMENT; or
else if (CONDITION) {STATEMENT; STATMENT2;}
  • Can only be attached after an "if" statement.
  • Is only checked if the previous conditions is false / falsy.
  • If the condition evaluate to true / truthy, than the statements are executed.
  • No limit of how many "else if" statements can be chained together.
 
const a = 7, b = 12;

if(a > b) // condition is false
    document.write("<p>a is bigger than b</p>");
else if(a < b) // condition is true
    document.write("<p>a is smaller than b</p>");
else statement (optional)
else STATEMENT; or
else {STATEMENT; STATEMENT2;}
  • Is the last statement, after "if" or "else if"
  • No condition, is only executed if no other condition is true / truthy
 
const a = 12, b = 7; 
                    
if(a < b) // condition is false
    document.write("<p>a is smaller then b</p>"); 
else // no true / truthy condition previous
    document.write("<a is bigger than b</p>");

switch statement

  • It is not possible to define a range for the case to match the condition
    e.g. case < 10 : or case >=10 && <=20 :
syntax explanation example
switch (CONDITION)
{		 The condition (value) to which the cases (values) are compared to. 
const country = prompt("Please, enter a country:"); 
let city;

switch(country)
    {
        case "italy":
           city = "Rom";    // city = "Rom"
           break;           // exit switch statement 
        case "england":     // city = "London"
        case "wales":       // city = "London"
        case "scotland":    // city = "London"
           city = "London"; 
           break;           // exit switch statement 
             default:       // if no case match
                city = "unknown";
      }
case VALUE1 : The value is compared to the condition.
STATEMENT1; One or more statements which are executed if the case matches the condition.
break; Is optional, at this point the parser exits the switch statement
case VALUE2 :
case VALUE3 :
case VALUE4 :
STATEMENT1-4;
break;		 Any of the "case values" which matches the "condition value" will trigger the following statement.
default:
STATEMENT;		 Is optional, if none of the cases matches, than the statement(s) after the default keyword are executed.
} closing bracket

Ternary Operator

CONDITION ? TRUE : FALSE

  • If the CONDITION evaluates to ...
  • ... true / truthy, the whole expression evaluates to the value after the ? ("TRUE")
  • ... false / falsy, the whole expression evaluates to the value after the : ("FALSE")

assigning value to variable depending on condition

VARIABLE = CONDITION ? VALUE1 : VALUE2 ;
// "condition" is ... true VALUE1 is assigned ... false VALUE2 is assigned

executing statement depending on condition

CONDITION ? STATEMENT1 : STATEMENT2 ;
// "condition" is ... true STATEMENT1 is executed ... false STATEMENT2 is executed

                let isNightTime = true; // true = light on / false = light off

                isNightTime ? console.log('Turn on the lights!') : console.log('Turn off the lights!');
