JavaScript reference

methods

Methods are functions stored as object properties.

window object (should be window.METHOD() but the first part 'window.' can be omitted)
syntax description example
prompt('STRING'); or
prompt('STRING1', 'STRING2');
  • ask user for input
  • stops the execution of the program till the user has reacted to the message
  • 'STRING1' message to the user
  • 'STRING2' place holder for the user input
  • user select OK, data from input field is returned as a string!
  • user select Cancel, the value null is returned
alert('STRING1');
  • display message to user ('STRING1')
  • stops the execution of the program till the user has reacted to the message
  • 'STRING1' message to the user
confirm('STRING');
  • Display a dialog with an optional message ('STRING'), and to wait until the user either confirms or cancels the dialog.
  • If the user selects OK, true is returned.
  • If the user selects Cancel or ignores the message, false is returned 
    const answer = confirm("Do you confirm?"); 
if(answer) // returns true
    document.write("You have confirmed.");
else       // returns false
    document.write("You have not confirmed.");

    The "window.confirm" method is executed,
    if the user has selected "OK", true is saved to const answer.
    if the user has selected "Cancel", false is saved to const answer.
