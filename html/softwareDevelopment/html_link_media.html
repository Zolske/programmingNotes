anchor attributes
href="   "
  • Must be set!
  • contains the path to where the link leads
    • external: another page
      • https://somewhere.com top of the page
      • https://somewhere.com#ID to the element with the id="ID"
    • internal: another part of the same page
      • # top of the page
      • #ID to the element with the id="ID"
    • opens the user's email program, ready to send a email to someone@example.com
      • mailto:someone@example.com
    • opens the users calling app, with the specified number +4733378901
      • tel:+4733378901
target="_blank"
  • opens the link in a new tab
download
  • is a boolean attribute (no value)
    • downloads the href="..." specified file download file
rel="   "
  • specifies the relationship between the current document and the linked document
  • search engines may use the information to determine how to index or display the page
    • help link to a help document
    • prev next link to previous / next document
    • nofollow links to an unendorsed document, like a paid link. ("nofollow" is used by Google, to specify that the Google search spider should not follow that link)
    • noopener prevents potential malicious behaviour from the destination document, which is useful for links to social media
    • external indicates that the referenced document is not part of the same site as the current document
