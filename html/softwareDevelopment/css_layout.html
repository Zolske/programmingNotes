the box model

Every HTML element is represented as a rectangular box on the page.

explain: the box model
  • parts of the box
  • content-box:
    • widthVALUE heightVALUE
    • %
      • percentage of the parent container
    • vw vh
      • percentage of the viewport size
    • rem
      • relative to the font size of the html
    • em
      • relative to the font-size of the parent element (can be difficult to control)
    • px
      • fixed value in pixel (can look different on screens with higher pixel density)
    • ch
      • size in characters

  • padding-box:

    Is the distance between "content-box" and "border".

    • paddingVALUE
    • single VALUE is used for all 4 sides, (sides can be set individually shorthand property)

    • padding-bottomVALUE , padding-leftVALUE , padding-rightVALUE , padding-topVALUE
    • sets the padding according the direction of the property
    • clears an area around the content box, the padding is transparent
  • border:
    • border-styleVALUE
    • single VALUE is used for all 4 sides, sides can be set individually with different styles (see shorthand property)
    • dotted
      dotted Defines a dotted border
    • dashed
      dashed Defines a dashed border
    • solid
      solid Defines a solid border
    • double
      double Defines a double border
    • groove
      groove Defines a 3D grooved border. The effect depends on the border-color value
    • ridge
      ridge Defines a 3D ridged border. The effect depends on the border-color value
    • inset
      inset Defines a 3D inset border. The effect depends on the border-color value
    • outset
      outset Defines a 3D outset border. The effect depends on the border-color value
    • none
      none no border

    • border-widthVALUE
    • single VALUE is used for all 4 sides, sides can be set individually with different thickness (see shorthand property)
    • thin , medium , thick
      • sets the border thickness respectively
    • LENGTH
      • sets the thickness to the specified thickness

    • border-colorVALUE
    • single VALUE is used for all 4 sides, sides can be set individually with different color (see shorthand property)

  • margin-box:
    • marginVALUE
    • single VALUE is used for all 4 sides, sides can be set individually with different lengths (see shorthand property)
    • margin-top , margin-right , margin-bottom , margin-left
      • sets the length for the sides individually
    • auto
      • the browser calculates the margin (horizontally center the element within its parent container)
    • LENGTH
      • specifies a margin in px, pt, cm, etc.
    • %
      • specifies a margin in % of the width of the containing element
    • -VALUE
      • negative values are allowed

    • margin collapsing:
    • the top and bottom margins of elements collapse into each other
      • e.g.: two elements on top of each other, the distance between each others border is the longest margin, not both margins together!
    • this does not happen on left and right margins! Only top and bottom margins!

    • clears the area outside the border, the margin is transparent
