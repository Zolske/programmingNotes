Typography

Typography is the art and technique of arranging type to make written language legible, readable and appealing when displayed.

fonts

Is the combination of typeface, size, weight, slope, and style to make up a printable or displayable set of characters. Font characters include letters, numbers, symbols, and punctuation marks.

font properties

Defines the font type.

font-family:
  • font-family:
  • no limit of how many values the property can have
  • browser tries from left to right to find a font it has installed
  • uses the first matching font
  • the last font should be a font from the "generic-font-family"

Generic Font Families

Are a group of fonts which should be used as "fallback" font.

explain: Generic Font Families
  • is a group of fonts which have the same characteristics
  • if no font is specified, than the browser uses the font which is set in its setting as "standard font"
  • setting the font as category of "Serif", "San-serif" or "Monospace", lets the browser use the font specified in its setting.
  • setting the font as category of "Cursive" or "Fantasy", lets the browser use a font which it has and fall into this category
  • "generic font families" are very useful as "fallback" fonts in case the browser dose not have the specified fonts but another with similar characteristics
  • Note: If the font name is more than one word, it must be in quotation marks, like: "Times New Roman".
Serif
fonts have a small stroke at the edges of each letter. They create a sense of formality and elegance.
example: serif

"Times New Roman", "Georgia", "Garamond"

Sans-serif
fonts have clean lines (no small strokes attached). They create a modern and minimalistic look.
example: sans-serif

"Arial", "Verdana", "Helvetica"

Note: On computer screens, sans-serif fonts are considered easier to read than serif fonts.

Monospace
all the letters have the same fixed width. They create a mechanical look.
example: monospace

"Courier New", "Lucida Console", "Monaco"

Cursive
fonts imitate human handwriting.
explain: cursive

"Brush Script MT", "Lucida Handwriting"

Fantasy
fonts are decorative/playful fonts.
explain: fantasy

"Copperplate", "Papyrus"
