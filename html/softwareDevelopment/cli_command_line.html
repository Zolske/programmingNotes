the command line

Command Line Interface (CLI) connects a user to a computer program or operating system. Through the CLI, users interact with a system or application by typing in text (commands). The command is typed on a specific line following a visual prompt from the computer.

  • basic commands:
  • clear
    • clears the terminal workspace (but not from history)
  • echo STRING
    • prints the string which follows the command to the terminal
      (e.g.: echo hello world prints hello world to the terminal)
  • history
    • shows a list of the last commands which can be called again by selecting its referencing number preceded by an !
      (e.g.: !3 recalls the 3th command from the history)
  • navigating the file and folder system
  • pwd (print working directory)
    • shows current directory
  • ls (list files)
    • lists all visible files in current directory
    • files are shown in regular text and folders in bold text
  • ls -la (list files -list all)
    • lists all (including hidden) files in current directory
  • cd .. change directory
    • moves one folder up
  • cd FOLDER_NAME
    • changes into specified folder
    • cd ~ changes back into home directory
    • cd / moves back to the root directory
  • working with files and folders:
  • touch FILE
    • creates a file with the specified name and file ending in the current directory
  • rm FILE (remove)
    • removes the specified file
  • rm -fl FOLDER (remove)
    • removes the specified folder including all its content and sub folders
  • mkdir FOLDER (makedirectory)
    • creates a new directory with the specified name in the current directory
  • cp FILE FOLDER (copy)
    • copies file into folder
  • cp FILE FILE_2 (copy)
    • copies file into same directory with the name of the second file
  • mv FILE FOLDER (move
    • moves file into folder
  • mv FILE FILE_2 (move
    • renames file into file_2
