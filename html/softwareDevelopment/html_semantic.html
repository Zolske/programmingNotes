Headings

Headings structure the page into a hierarchical order and describes the content of the article.

  • <h1> - <h6>
  • only one <h1> per page which is the title of the page, "SEO" may use it for there search results
  • screen reader use headings to navigate the page
  • no heading level skipping from <h2> to <h6>, but from <h6> to <h2> possible

paragraph

Contain the text of the page.

  • <p>
  • Begins a new idea or point. An original idea should always start with a new paragraph.
  • Provide a pause for readers.
  • Separate contrasting information or ideas.
  • Complete an introduction or start a conclusion.
